class pyqir.parser.QirCondBrTerminator(term: PyQirTerminator)¶

Bases: QirTerminator

Instances of QirCondBrTerminator represent a conditional branch terminator instruction that decides which named block to jump to based on an given operand.

property condition: QirOperand¶

Gets the QirOperand representing the condition used to determine the block to jump to.

property false_dest: str¶

Gets the name of the block that will be jumped to if the condition evaluates to false.

property true_dest: str¶

Gets the name of the block that will be jumped to if the condition evaluates to true.

class pyqir.parser.QirSwitchTerminator(term: PyQirTerminator)¶

Bases: QirTerminator

Instances of QirSwitchTerminator represent a switch terminator instruction that can jump to one or more blocks based on matching values of a given operand, or jump to a fallback block in the case that no matches are found.

property default_dest: str¶

Gets the name of the default block that the switch will jump to if no values match the given operand.

property dest_pairs: List[Tuple[QirConstant, str]]¶

Gets a list of pairs representing the constant values to compare the operand against and the matching block name to jump to if the comparison succeeds.

property operand: QirLocalOperand¶

Gets the operand variable of the switch statement.

class pyqir.parser.QirUnreachableTerminator(term: PyQirTerminator)¶

Bases: QirTerminator

Instances of QirUnreachableTerminator represent an unreachable terminator instruction. As the name implies, this terminator is not expected to be reached such that some instruction in the block before this terminator should halt program execution.

class pyqir.parser.QirInstr(instr: PyQirInstruction)¶

Bases: object

Instances of QirInstr represent an instruction within a block of a QIR program. See the subclasses of this type for specifically supported instructions.

property output_name: Optional[str]¶

Gets the name of the local operand that receives the output of this instruction, or None if the instruction does not return a value.

property type: QirType¶

Gets the QirType instance representing the output of this instruction. If the instruction has no output, the type will be an instance of QirVoidType.

class pyqir.parser.QirPhiInstr(instr: PyQirInstruction)¶

Bases: QirInstr

Instances of QirPhiInstr represent a phi instruction that selects a value for an operand based on the name of the block that transferred execution to the current block.

get_incoming_value_for_name(name: str) → Optional[QirOperand]¶

Gets the QirOperand representing the value for a given originating block, or None if that name is not found. :param name: the block name to search for.

property incoming_values: List[Tuple[QirOperand, str]]¶

Gets a list of all the incoming value pairs for this phi node, where each pair is the QirOperand for the value to use and the string name of the originating block.
